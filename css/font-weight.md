# `font-weight`

## Numerical values

```css
/* Thin */
font-weight: 100;

/* Extra Light */
font-weight: 200;

/* Light */
font-weight: 300;

/* Normal */
font-weight: 400;

/* Medium */
font-weight: 500;

/* Semi Bold */
font-weight: 600;

/* Bold */
font-weight: 700;

/* Extra Bold */
font-weight: 800;

/* Black */
font-weight: 900;
```

## Alias values

```css
/* Equivalent to `400` */
font-weight: normal;

/* Equivalent to `700` */
font-weight: bold;
```

## Relative values

```css
/* Lighter weight than the inherited value */
font-weight: lighter;

/* Bolder weight than the inherited value */
font-weight: bolder;
```
