# Truthy and falsy values

## Falsy values

- `false`
- `0`
- `''` (empty string)
- `undefined`
- `null`
- `NaN`

> [`Falsy` on MDN]
(https://developer.mozilla.org/en-US/docs/Glossary/Falsy).

## Truthy values

*Everything else.*

> [`Truthy` on MDN]
(https://developer.mozilla.org/en-US/docs/Glossary/Truthy).
